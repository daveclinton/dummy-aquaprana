# Dummy Admin

Static admin dashboard ready for GitHub Pages hosting.

## Local preview

Open `index.html` in a browser.

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`, set `Source` to `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save and wait for Pages to publish.

## Custom domain

1. In `Settings` -> `Pages`, enter your domain in `Custom domain`.
2. Add the required DNS records at your domain registrar.
3. Wait for DNS to propagate, then enable `Enforce HTTPS` when GitHub makes it available.
